# CombatLog

Lightweight configurable combat logging prevention plugin for Paper servers.

CombatLog prevents players from escaping fights by logging out.
Designed to be simple, fast, and compatible with most plugins.

Players who leave during combat will receive a configurable penalty.

---

## Features

- Configurable combat duration
- PvP / PvE combat detection toggle
- Multiple logout penalties
  - Kill
  - Kick
  - Clear Inventory
  - Drop Inventory
- BossBar combat indicator
- Combat start & end messages
- Permission bypass support
- Reload without server restart

---

## Commands

/combat reload  
/combat duration <seconds>

---

## Permission

combat.bypass

---

## Supported Versions

Paper 1.20+

# 한국어 안내

가볍고 설정 가능한 전투 로그 방지 플러그인입니다.

플레이어가 전투 중 접속 종료를 통해 전투를 회피하지 못하도록 설계되었습니다.  
다른 플러그인과의 충돌을 최소화하고, 서버 성능에 부담을 주지 않는 것을 목표로 합니다.

전투 중 로그아웃 시 설정된 처벌이 적용됩니다.

## 주요 기능

- 전투 지속 시간 설정
- PvP / PvE 전투 감지 선택
- 다양한 로그아웃 처벌
  - 즉사
  - 강퇴
  - 인벤토리 삭제
  - 인벤토리 드롭
- 보스바 전투 표시
- 전투 시작/종료 메시지
- 권한 면제 지원
- 서버 재시작 없이 설정 리로드

## 명령어

/combat reload  
/combat duration <초>

## 권한

combat.bypass
